<strong>APEX PORTFOLIO THEME</strong>
<a href="http://instructionaltechnology.wooster.edu">APEX Portfolio Theme</a>

<strong>INSTALL</strong>
1. Upload the APEX Portfolio child theme folder via FTP to your wp-content/themes/ directory. (The Genesis parent theme needs to be in the wp-content/themes/ directory as well.)
2. Go to your WordPress dashboard and select Appearance.
3. Activate the APEX Portfolio Theme.

<strong>WIDGET AREAS</strong>
This is a full width theme and does not support sidebars or widget areas.

<strong>FEATURED IMAGES</strong>
By default WordPress will create a default thumbnail image for each image you upload and the size can be specified in your dashboard under Settings > Media.
