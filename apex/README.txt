<strong>APEX THEME</strong>
<a href="http://apex.wooster.edu">APEX Theme</a>

<strong>INSTALL</strong>
1. Upload the APEX child theme folder via FTP to your wp-content/themes/ directory. (The Genesis parent theme needs to be in the wp-content/themes/ directory as well.)
2. Go to your WordPress dashboard for the main site of the Network and select Appearance.
3. Activate the APEX theme.

<strong>WIDGET AREAS</strong>
The theme is a full width theme and does not support the display of sidebars or widget areas.

<strong>FEATURED IMAGES</strong>
By default WordPress will create a default thumbnail image for each image you upload and the size can be specified in your dashboard under Settings > Media.
